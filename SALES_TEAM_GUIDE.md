# 영업단 사용 가이드

## 📋 개요

각 영업 담당자는 자신만의 개인 홈페이지를 가지며, 링크를 공유하여 고객에게 자신을 소개할 수 있습니다.

---

## 🔐 로그인이 필요한가요?

### 편집을 위해서는 **로그인이 필요**합니다

- **본인의 페이지 편집**: 로그인 필요 ✅
- **다른 사람의 페이지 보기**: 로그인 불필요 ❌
- **링크로 접속한 고객**: 로그인 불필요, 보기만 가능 ❌

---

## 🔗 링크 공유 및 접근 권한

### 시나리오 1: 본인이 자신의 페이지 편집
1. 로그인 (`/login`)
2. 홈페이지 접속 → 자동으로 자신의 페이지로 이동
3. "편집" 버튼 클릭
4. 이름, 소개글, 프로필 이미지 수정
5. 저장 후 변경사항 반영

### 시나리오 2: 고객이 링크로 접속
1. 링크 받기: `https://yourdomain.com/user/sanghun`
2. 링크 클릭 → 해당 담당자의 페이지 표시
3. **편집 불가** (보기 전용)
4. 로그인하지 않은 상태로도 볼 수 있음

### 시나리오 3: 다른 담당자의 페이지 보기
- 로그인 없이도 다른 담당자의 페이지 확인 가능
- 편집 불가 (본인만 편집 가능)

---

## 👥 관리자: 사용자 추가 방법

### 방법 1: 개별 사용자 추가 (추천)

1. 관리자 페이지 (`/admin`) 접속
2. "개별 사용자 추가" 섹션에서 "+ 사용자 추가" 클릭
3. 정보 입력:
   - **이메일**: 로그인에 사용 (예: `sanghun@company.com`)
   - **사용자명**: URL에 사용 (예: `sanghun`)
   - **이름**: 표시될 이름 (예: `김상훈`)
   - **비밀번호**: 초기 비밀번호 설정
4. "사용자 등록하기" 클릭

**등록된 사용자에게 알려줄 정보:**
- 이메일 (로그인용)
- 초기 비밀번호
- 자신의 페이지 URL: `https://yourdomain.com/user/[사용자명]`

### 방법 2: 일괄 등록 (여러 명 한번에)

1. 관리자 페이지 접속
2. "영업단 일괄 등록" 섹션에서 "+ 일괄 등록" 클릭
3. CSV 형식으로 입력:

```
이메일,사용자명,이름,비밀번호
sanghun@company.com,sanghun,김상훈,temp123!
minji@company.com,minji,이민지,temp123!
junho@company.com,junho,박준호,temp123!
```

4. "일괄 등록하기" 클릭
5. 결과 확인 (성공/실패 표시)

## 📝 각 담당자 사용 방법

### 1. 처음 로그인하기

1. `/login` 페이지 접속
2. 관리자가 알려준 이메일과 비밀번호로 로그인
3. 처음 로그인 후 비밀번호 변경 권장 (추가 예정)

### 2. 자신의 페이지 편집하기

1. 로그인 후 홈페이지 접속
2. 자동으로 자신의 페이지(`/user/[자신의사용자명]`)로 이동
3. 우측 상단 "편집" 버튼 클릭
4. 수정 가능 항목:
   - **이름**: 표시될 이름
   - **소개글**: 본인 소개 (줄바꿈 가능)
   - **프로필 이미지 URL**: 프로필 사진
5. "저장하기" 클릭

### 3. 링크 공유하기

1. 자신의 페이지에서 URL 복사:
   - 예: `https://yourdomain.com/user/sanghun`
2. 고객에게 링크 공유
3. 고객은 로그인 없이 페이지 확인 가능
4. **고객은 편집 불가** (보기 전용)

## 🔒 권한 체계

| 상황 | 로그인 필요 | 편집 가능 |
|------|------------|----------|
| 본인의 페이지 편집 | ✅ 필요 | ✅ 가능 |
| 본인의 페이지 보기 | ❌ 불필요 | - |
| 다른 사람의 페이지 보기 | ❌ 불필요 | ❌ 불가능 |
| 링크로 접속한 고객 | ❌ 불필요 | ❌ 불가능 |
| 관리자 | ✅ 필요 | ✅ 모든 페이지 편집 가능 |

## 📋 체크리스트

### 관리자 체크리스트
- [ ] 모든 영업 담당자 정보 수집 (이메일, 이름, 사용자명)
- [ ] 관리자 페이지에서 사용자 등록
- [ ] 각 담당자에게 로그인 정보 전달 (이메일, 비밀번호)
- [ ] 각 담당자에게 자신의 페이지 URL 안내

### 담당자 체크리스트
- [ ] 로그인 정보 확인
- [ ] 로그인 후 자신의 페이지 확인
- [ ] 프로필 편집 (이름, 소개글, 이미지)
- [ ] 자신의 페이지 링크 저장 및 공유 준비

---

## 💡 팁

1. **비밀번호 변경**: 보안을 위해 처음 로그인 후 비밀번호 변경 권장
2. **프로필 이미지**: 이미지 URL을 사용하거나, 이미지 호스팅 서비스 활용
3. **소개글**: 줄바꿈을 원하면 엔터를 눌러서 작성 (줄바꿈 유지됨)
4. **링크 공유**: 명함, 이메일 서명, SNS 등에 자신의 페이지 링크 추가

## ❓ 자주 묻는 질문

**Q: 로그인하지 않아도 링크로 볼 수 있나요?**
A: 네, 링크로 접속한 고객은 로그인 없이도 페이지를 볼 수 있습니다.

**Q: 다른 사람이 내 페이지를 편집할 수 있나요?**
A: 아니요. 본인만 편집할 수 있습니다. 링크로 접속한 사람은 보기만 가능합니다.

**Q: 비밀번호를 잊어버렸어요.**
A: 관리자에게 문의하여 비밀번호 재설정을 요청하세요.

**Q: 사용자명을 변경할 수 있나요?**
A: 사용자명은 URL에 사용되므로 변경 불가능합니다. 관리자에게 문의하세요.
