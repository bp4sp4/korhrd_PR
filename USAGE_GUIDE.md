# 사용 가이드

## 🚀 시작하기

### 1. 개발 서버 실행

```bash
cd korhrdmkt
npm run dev
```

브라우저에서 `http://localhost:3000` 접속

---

## 📖 주요 기능 사용법

### 1️⃣ 홈페이지 (메인 페이지)

**URL**: `http://localhost:3000`

- 모든 사용자 목록을 카드 형태로 확인
- 각 사용자 클릭 시 개인 페이지로 이동
- 로그인/로그아웃 버튼
- 관리자는 우측 상단에 "관리자" 버튼 표시

---

### 2️⃣ 사용자 개인 페이지

**URL**: `http://localhost:3000/user/[사용자명]`

**예시**: `http://localhost:3000/user/sanghun`

#### 기능:
- 프로필 이미지, 이름, 사용자명, 소개글 표시
- 본인이 로그인한 경우 → "편집" 버튼 표시
- 다른 사용자의 페이지 → 읽기 전용

---

### 3️⃣ 프로필 편집

**URL**: `http://localhost:3000/user/[사용자명]/edit`

**접근 방법**:
1. 자신의 프로필 페이지에서 "편집" 버튼 클릭
2. 또는 직접 URL 입력

#### 편집 가능 항목:
- ✅ 이름
- ✅ 소개글
- ✅ 프로필 이미지 URL

#### 권한:
- 본인만 자신의 페이지 편집 가능
- 관리자는 모든 페이지 편집 가능

---

### 4️⃣ 로그인

**URL**: `http://localhost:3000/login`

#### 로그인 방법:
1. 사용자명 입력 (현재는 비밀번호 불필요 - 임시)
2. "로그인" 버튼 클릭
3. 로그인 성공 시 홈으로 이동

#### 기본 계정:
- **일반 사용자**: `sanghun`
- **관리자**: `admin`

---

### 5️⃣ 관리자 페이지 ⭐

**URL**: `http://localhost:3000/admin`

**접근 권한**: 관리자 계정으로 로그인한 경우만

#### 관리자 기능:

##### 📊 통계 대시보드
- 전체 사용자 수
- 일반 사용자 수
- 관리자 수

##### 👥 사용자 관리
- **사용자 목록 조회**: 모든 사용자 정보 테이블로 표시
  - 프로필 이미지
  - 이름
  - 사용자명 (클릭 시 해당 사용자 페이지로 이동)
  - 권한 (일반/관리자)
  - 생성일

##### ➕ 사용자 추가
1. "사용자 추가" 버튼 클릭
2. 정보 입력:
   - 사용자명 (예: `sanghun`)
   - 이름 (예: `상훈`)
   - 권한 선택 (일반 사용자 / 관리자)
3. "생성하기" 버튼 클릭

##### 🗑️ 사용자 삭제
- 각 사용자 행의 "삭제" 버튼 클릭
- 본인은 삭제 불가

## 📝 사용 시나리오

### 시나리오 1: 새 사용자 추가 (관리자)

1. `admin` 계정으로 로그인
2. `/admin` 페이지 접속
3. "사용자 추가" 버튼 클릭
4. 정보 입력 후 생성
5. 새 사용자가 홈페이지와 목록에 표시됨

### 시나리오 2: 사용자가 자신의 프로필 편집

1. 일반 사용자 계정으로 로그인 (예: `sanghun`)
2. `/user/sanghun` 페이지 접속
3. "편집" 버튼 클릭
4. 이름, 소개글, 이미지 URL 수정
5. "저장하기" 클릭
6. 변경사항이 프로필 페이지에 반영됨

### 시나리오 3: 여러 사용자 프로필 확인

1. 로그인 없이 홈페이지 접속
2. 각 사용자 카드 클릭하여 프로필 확인
3. 읽기 전용으로 모든 프로필 확인 가능

## 💾 데이터 저장

현재는 **브라우저의 localStorage**에 저장됩니다.

- 브라우저 개발자 도구 → Application → Local Storage에서 확인 가능
- 키: `korhrd_users` (사용자 데이터)
- 키: `korhrd_auth` (세션 정보 - sessionStorage)

> ⚠️ 주의: 다른 브라우저나 시크릿 모드에서는 데이터가 공유되지 않습니다.

---

## 🔐 권한 체계

### 일반 사용자 (`user`)
- ✅ 자신의 프로필 페이지 확인
- ✅ 자신의 프로필 편집
- ✅ 다른 사용자 프로필 읽기
- ❌ 다른 사용자 프로필 편집 불가
- ❌ 관리자 페이지 접근 불가

### 관리자 (`admin`)
- ✅ 모든 사용자 프로필 확인
- ✅ 모든 사용자 프로필 편집
- ✅ 관리자 페이지 접근
- ✅ 사용자 생성/삭제
- ✅ 사용자 권한 확인

---

## 🛠️ 문제 해결

### 문제: 로그인이 안 됨
- 사용자명이 올바른지 확인
- 브라우저의 개발자 도구 콘솔에서 에러 확인

### 문제: 편집 버튼이 안 보임
- 로그인이 되어 있는지 확인
- 본인의 페이지인지 확인

### 문제: 관리자 페이지 접근 불가
- 관리자 계정(`admin`)으로 로그인했는지 확인
- 로그아웃 후 다시 로그인 시도

### 문제: 데이터가 저장되지 않음
- 브라우저가 localStorage를 지원하는지 확인
- 시크릿 모드가 아닌지 확인
